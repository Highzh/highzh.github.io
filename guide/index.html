# 学习概论

学习顾名思义包含两个部分，一个是学，一个是习

# 学习的学

学习学的什么？学的都是科学文化知识，实际上就是前人的经验教训，我们简称经验，经验经过去伪存真、一般化之后就是知识，知识经过反复验证，多次重复之后依然正确的叫做科学。

知识怎样学习才是最佳的，这其实是一个伪命题，因为：知识不是学出来的而是用出来的，听到的看到的永远都不是自己的，只有经过实践（纵向提升）+扩散（横向扩展），在生活中不断的应用和融会贯通才算是真正的学到的自己的经验。

学习是获得知识经验的过程，那有什么要求吗？

知识的 获取需要已有的知识经验作为基础，不知道字怎么读就无法阅读书籍领会文章的思想，不知道什么是代数就无法对代数式进行运算，不知道英文单词的意思就听不懂歪果仁说话，不知道全国省市的简称就无法通过车牌号判断车辆归属地，等等。

有些简单的知识只需要常识便可以理解学习的并不是不需要知识，常识也是知识，只是最简单而已。

因此我们可以认为：将已有的知识与未知的知识建立联系的过程就是学习。

而建立联系的方法就叫学习方法。

今天我们不谈方法，只说学习。

既然学习就是将已有的知识与未知的知识建立联系，那一个学足以表达了为什么还要习呢？

因为人的记忆是有限且短暂的，这样建立的联系就不是牢固的，需要反复的实践和重复。

# 学习的习

其实学习的重点在习，而并非学上面，因为学是不需要付出什么 的，看一看、听一听就是学。

但是很多知识反复看，反复听但是在自己独立完成的时候仍然是无法做到的，因为学自然是学了，只是并没有“会”，这就是学不会。

要想学会就必须反复的习。

现在让我们回首小学时候自己语文课后经常要写很多遍生字，数学课后最简单的数字还要写好多遍。正是自己反复的练习才有了今天不假思索就能写出来自己想写的字，自己想计算的数学式子。

当基础牢固了之后就上了中学，中学时候还是要大量的练习，小学时候哭着写作业的情况不会出现了，因为长大了学会了坚强，当然也有可能学会了偷懒，这个时候成绩就出现了较大差异，有的同学成了别人家的孩子，有的同学天天被教育向别人家的孩子学习。

其实别人家的孩子没什么值得学习，除了勤奋的勤，因为智商、习惯都是很难学习的，但是恰恰也是比较不影响成绩的，况且，智商又能差别多少呢？

从另外一个方面来说：我们能学习的也只有勤奋，因为每个人都是独立的人，价值观和性格不同，习惯也必然不同，别人的习惯说到底也是别人的经验，每个人独有的经验可以借鉴，但绝不能照抄。
